This project is a static frontend website designed for Sai Pranavi Hostel.
It provides users with information about:

Hostel rooms 🛏️
Facilities 🏠
Contact details 📞
🚀 Features
🏠 Home Section – Introduction of hostel
🛏️ Rooms Section – Types of rooms available
🏢 Facilities Section – Amenities provided
📷 Gallery Section – Images of hostel
📞 Contact Section – Phone / address details
📱 Responsive Design – Works on mobile & desktop
🛠️ Tech Used
HTML → Structure
CSS → Styling & layout
JavaScript → Basic interactivity
